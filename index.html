Our Thoughts

Our thoughts is a full stack app built in the MEAN stack. It allows users to post thoughts/entries that they have. It then uses the Twilio API to send a text message of the entrie to users. The main objective of this App was to create an App with authorization, and integrating in the Twilio API.
